# YouTube 음악 Player

qt 라이브러리를 이용하여 YouTube 음원 스트리밍 / 재생목록 설정을 할 수 있는 프로그램



## 주요 기능 

1. 다양한 재생 목록을 기반으로 음악 저장

2. 음악은 YouTube 링크를 바탕으로 추가

3. 음악 재생시 썸네일, 조회수 등의 정보 표출



## 개발 진행 상황

* 프로그램의 레이아웃, 구조 설계 완료
